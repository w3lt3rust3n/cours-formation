Ce site est conçu pour être adaptable sur trois résolutions spécifiques:

  • Une résolution écran de 1000px de large;
  • Une résolution tablette de 501px de large;
  • Une résolution smartphone de 320px de large.

La bannière est paramétrée dans le background-image du header, le fichier source change donc en fonction de la largeur de l'écran.

La couleur de fond de la boîte globale change en fonction de la résolution.

La police utilisée pour le texte est différente dans les 3 versions: la police version smartphone est légèrement plus grande, ce qui permet une meilleure lisibilité sur un petit écran.
